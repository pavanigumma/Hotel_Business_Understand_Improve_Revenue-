## Project Description
This Power BI project focuses on analyzing hotel business performance to understand revenue drivers, occupancy trends, and guest behavior. 
The dashboard provides interactive insights to support data-driven decision-making for improving profitability and customer segmentation.
##Objectives
- Analyze hotel revenue and occupancy performance
- Track key performance indicators (KPIs)
- Understand booking trends over time
- Compare booking sources (Direct vs OTA)
- Segment guests into meaningful customer groups
## Key Performance Indicators (KPIs)
- **Total Revenue**
- **Occupancy Percentage**
- **Average Daily Rate (ADR)**
- **Revenue per Available Room (RevPAR)**
- **Net Profit**
## Analysis & Visualizations
- **Booking Trends**
  - Daily, Monthly, and Seasonal analysis
- **Booking Source Comparison**
  - Direct bookings vs Online Travel Agencies (OTA)
- **Guest Analysis**
  - Nationality-wise guest distribution
  - Booking source analysis
  - Length of stay analysis
- **Customer Segmentation**
  - First-time Guests
  - Loyal Guests
  - High Spenders
- **Interactive Filters**
  - Room Type
  - Hotel Branch / Location
  - Booking Source
##  Dashboard Features
- KPI cards for quick performance overview
- Interactive slicers for dynamic filtering
- Clear and intuitive layout for business users
- Visual insights for revenue optimization
